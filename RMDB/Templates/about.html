What is RMDB?

My name is Reid Harrington, I am a first year computer science student at Laurier University. This is a personal project I have been working on in my spare time. It takes a text document filled with a list of movies, and a python script converts the movies into movie objects. This website displays the python movie objects and uses a movie database API for the posters.
